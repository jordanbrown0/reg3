Global Configuration

General Stuff

  • Convention name - displayed in UI and on reports
  • Convention start date - memberships entered before this are considered preregistered.
  • Currency prefix, suffix - symbols to display before and after currency amounts, e.g. "$", "CDN$", "FF".
  • Screen font size, in points (give or take). Note that this is a global setting; it is not intended as a personal preference.

    • Badges

  • Badge font - unfortunately, you must type it and type it correctly.
  • Number of copies - of each label to print.
  • Real name too? - When member has a badge name, print a label with badge name and another with real name.
  • City? Size? - Whether to print the city, and what size (in pixels) to print it.
  • Number? Size? - Whether to print the number, and what size (in pixels) to print it.
  • Sizes to try for names - sizes (in pixels), in decreasing order, to try when printing the member's name. Use and to add and remove entries.
  • Margins: Left, Right, Top, Bottom - reserves space on the sides of the badge, mostly to allow for printer misalignment.

    • Advanced

  • How often to check configuration - when a configuration change is made on another station.
