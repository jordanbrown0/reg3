Add/Edit custom field definition

Add the definition of a new set of fields, or edit an existing definition.

  • number of the member data page to add these fields to. Can be one of the built-in pages, or a larger number to add a new page.
  • if defining a new page, this gives the page title. If adding to an existing page, this value is not used but can help to remind you which page you are adding to.
  • a list of fields to add. Use the and buttons to add or remove entries.

    • For each field:

    • name of new field.
    • label to show on edit pages.
    • type of data that goes in this field
