Station Configuration

General

  • Station name - typically a label on the station.
  • Server name - that the station is attached to (read only).
  • Label printer - that this station will use.
  • Metaclasses - if not empty, this station is only allowed to issue these metaclasses. See membership class edit.

    • Offline Operations

  • Print badges now - disable if you want to mark somebody checked in without printing a badge.
  • Mark badges picked up - disable if you want to print badges without marking the member picked up.
  • Use current time for records - disable if you want to mark members entered or picked up as of another date and time.
  • If not, keep records as of - what date and time to use for records.

    • Permissions

  • Add new member
  • Administration
  • Reports
  • Process upgrades
  • Ad hoc upgrades (requires "Process upgrades")
  • Process transfers
  • Mark membership as not-picked-up
  • Void / unvoid memberships

    • Note that disabling the Administration menu means that you cannot use the entry on the main menu to get back to this form to re-enable it. However, you can use to get to the Administration menu even if the menu entry is disabled.
