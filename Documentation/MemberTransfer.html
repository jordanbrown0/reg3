Transfer Membership

Enter information for the new owner of the membership. When you are done, to return to the member display page. to return to the main menu without transferring the membership.

Use to use the same address as the member being transferred from.

Note: The name and address are cleared in preparation for the new owner's information. Other fields are not. The values in those other fields may not be appropriate for the new owner and must be updated as required.

Note: the old owner's information is retained, but marked as "transferred". The new owner gets a new record.
