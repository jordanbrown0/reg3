Inspect member information

Inspect member information, then use one of the buttons to operate on the entry.
- Make changes, switching to the member edit page.
- Print a label, mark the badge as picked up, and return to the main menu.
- Upgrade to another membership class, switching to the upgrade picker page.
- Transfer to another person, switching to the member transfer page.
- Mark the membership as void.
- If the badge was already picked up, mark it as not-picked-up.
- If the membership has been marked void, return it to normal.
