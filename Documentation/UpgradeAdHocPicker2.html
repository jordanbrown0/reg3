Ad hoc upgrade details

Here you give the details of an ad hoc upgrade, an upgrade for which there is no standard policy. Most of the information on this page is read-only; it tells you what you have proposed to do and what has gone before.

  • From class - member's current class
  • To class - class member is to be changed to
  • Paid so far - how much the member has paid so far
  • Class cost - how much this class normally costs
  • Upgrade cost - how much you are going to charge (or, if negative, refund) for this upgrade

    • When you are done, saves the change and returns you to the member display. returns you to the main menu without upgrading.
