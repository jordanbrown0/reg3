Add/Edit correction definition

Add the definition of a new set of corrections, or edit an existing definition.

  • table that the correction applies to. At this time, this should always be "members".
  • name of the field that the correction applies to. (Should really be a picker, ref issue #416.)
  • a list of from/to corrections. Use the and buttons to add or remove entries.
    • When editing the specified field, matching values are automatically replaced when you leave the field. "From" values are matched case-insensitively. (Note that this makes a "US"->"US" entry correct "us" to "US".)
