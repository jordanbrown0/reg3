Membership summary report

This report presents various membership statistics.
  • Warm bodies - number of memberships picked up
  • No show - number of preregistered memberships not picked up
  • Preregistered - number of memberships entered before the convention
  • At the door - number of memberships entered at the convention
  • Grand total - total number of memberships (not including transfered and void)
  • Paid - number of paid memberships
  • Free - number of free memberships
  • Transferred - number of memberships transferred from one person to another
  • Void - number of memberships voided
